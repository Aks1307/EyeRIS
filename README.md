# EyeRIS

[Demo video][youtube]

[youtube]: https://www.youtube.com/watch?v=9ZHfn0uBxb8&feature=youtu.be

Predictive mental health app collecting audio data and raw EEG data using the Muse headband. This project won 2nd place at HackMentalHealth 2018.
